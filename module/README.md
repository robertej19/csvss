# cssplt

Single-file HTML+CSS-only interactive plots (no JavaScript).

**Step 1:** Skeleton; run `python examples/demo_linked.py` to generate `demo_linked.html`.

**Demo data:** Run `python examples/generate_demo_data.py` to create `examples/data/demo_heatmap.csv`, `demo_kde.csv`, and `demo_radar.csv` for sample plots.
